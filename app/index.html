Remote debugger

Для запуска скрипта добавьте в BODY: 

<script src="http://:/rd.js" type="text/javascript"></script>

и создайте в корне приложения файл с настройками:

top.config = {
  'server' : {
    'ip' : '192.168.0.2',
    'port' : 7760
  }
};

Для отправки сообщения используйте:

rd.log('MESSAGE');
rd.info('MESSAGE');
rd.error('MESSAGE');
rd.warn('MESSAGE');
